# PeerFinder - Telegram-бот для поиска участников проектов Школы 21

Бот помогает находить людей для командных проектов и поиска проверяющих на платформе [21-school.ru](https://platform.21-school.ru).

Ты указываешь id проекта, выбираешь кластер кампусов и фильтр по статусу - бот возвращает список участников.

## Возможности

- Авторизация через JWT-токен или логин/пароль от платформы
- Выбор кластера кампусов: Казань/Москва/Новосибирск, Ташкент/Самарканд, остальные
- Поиск участников по id проекта
- Фильтр по статусу участия: назначен, зарегистрирован, в процессе, на проверке, принят, провалил
- Постраничный вывод результатов (25 человек на страницу)

## Требования

- CMake ≥ 3.14
- Компилятор с поддержкой C++20
- OpenSSL
- Boost

Зависимость [tgbot-cpp](https://github.com/reo7sp/tgbot-cpp) подтягивается автоматически через CMake FetchContent.

## Сборка

```bash
git clone https://github.com/TheMidnightTraveler/s21-peer-finder
cd PeerFinder
cmake -B build
cmake --build build
```

## Запуск

Токен бота передаётся через переменную окружения `BOT_TOKEN`:

```bash
export BOT_TOKEN="ваш_токен_от_BotFather"
./build/s21_bot
```

Токен можно получить у [@BotFather](https://t.me/BotFather) командой `/newbot`.

## Использование

1. `/start` - начать работу, выбрать способ авторизации
2. Авторизоваться одним из способов:
   - **JWT токен** - вставить токен, полученный с платформы
   - **Логин и пароль** - ввести учётные данные от аккаунта Школы 21
3. `/search` - начать поиск:
   - Выбрать кластер кампусов
   - Ввести id проекта (числовой, из URL страницы проекта)
   - Выбрать статус участников
4. Пролистывать результаты кнопками ◀️ / ▶️
